## Descrição

Repositório inicial do framework TypeScript [Nest](https://github.com/nestjs/nest).

## Instalação

```bash
$ npm install
```

## Executando a aplicação

```bash
# desenvolvimento
$ npm run start

# modo de observação
$ npm run start:dev

# modo de produção
$ npm run start:prod
```

## Teste

```bash
# testes unitários
$ npm run test

# testes e2e
$ npm run test:e2e

# cobertura de testes
$ npm run test:cov
```
